Mac

  • Change directory - cd PATH
    • ie c:/folder/anotherFolder

  • Create a folder - mkdir folderName
    • ie: testFolder

  • Create a file - touch fileName
    • ie: touch myTestFile

  • Delete a folder and all contents - rm -R PATH
    • ie: c:/folder

  • Delete a file - rm fileName
    • ie: test.txt
